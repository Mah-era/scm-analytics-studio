Supply Chain Analytics Prototype

SCM Maker | SCM Analytics Studio

A self-initiated local-first prototype for supply chain analytics, dashboarding, KPI tracking, data cleaning, and decision-support workflows across demand, inventory, procurement, logistics, warehouse, production, and cost operations.

View GitHub Repository Read Project Documentation

What It Demonstrates

Practical SCM process thinking through dashboards, KPI formulas, sample datasets, mapping flows, validation checks, and structured analysis modules.

Core Capabilities

  • CSV/XLSX upload and column mapping
  • Interactive SCM dashboards and filters
  • Forecast, inventory, supplier, logistics, warehouse, MRP, and cost analytics
  • Local API, MCP-style server, CLI workflows, and plugin-style skills

Project Scope

This page is a static project overview. The analytics system itself is a local Streamlit prototype and is not presented as a launched enterprise product.

Built to explore how digital operations tools can improve visibility, structure, and decision support across supply chain management workflows.
