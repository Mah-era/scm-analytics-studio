# Sample Data Guide

Use `sample_scm_data.xlsx` to test the full app.

Recommended sheet:
- `Integrated_SCM_Data`: contains all major fields for all dashboards.

Special training files:
- `messy_scm_training_data.csv`: intentionally includes missing values, duplicates, mixed dates, inconsistent category labels, and currency-like numeric strings.
- `training_situations_catalog.csv`: explains each training scenario and expected behavior.

Sheets:
- Demand_Sales
- Inventory
- Procurement
- Logistics
- Warehouse
- Production
- Cost_Profitability
- Data_Issues_Training
